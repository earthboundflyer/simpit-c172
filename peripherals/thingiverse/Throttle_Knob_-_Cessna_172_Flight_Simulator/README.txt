# Summary

A throttle knob for a Cessna 172 flight simulator panel. I purchased a genuine throttle for my simulator project and I modeled this from the calipered dimensions of the throttle for others to use.

This knob is modeled from the [Aircraft Spruce A-820 friction lock control](https://www.aircraftspruce.com/catalog/appages/a820.php).

For more information about the project, see the details at https://cessna172sim.allanglen.com
