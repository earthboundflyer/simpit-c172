# Summary

A mixture knob and button for a Cessna 172 flight simulator panel. I purchased a genuine Vernier mixture control for my simulator project but I modeled this to the calipered dimensions of the real mixture control ([Aircraft Spruce A-790](https://www.aircraftspruce.com/catalog/appages/a7902.php)) for others to use.

The preview .stl file is provided only for a preview of the knob and button as they would be aligned when assembled. The button travel in the real mixture control is about 3mm.

For more information about the project, see the details at https://cessna172sim.allanglen.com
